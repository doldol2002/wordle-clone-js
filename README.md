# Wordle Clone (JavaScript)

A browser-based implementation of the popular Wordle word-guessing game built using HTML, CSS, and JavaScript.

## Overview

This project demonstrates string comparison algorithms, dynamic DOM manipulation, and state management in a front-end application without external libraries.

## Features

* Random 5-letter word selection
* 6 guess attempts
* Letter-by-letter validation
* Color-coded feedback (correct position, wrong position, incorrect)
* Keyboard input handling

## Technical Design

* Target word selected from predefined dictionary array
* Guess evaluation algorithm compares user input to target word
* Game state managed using arrays
* DOM updated dynamically after each guess

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript (ES6)

## Future Improvements

* Larger validated dictionary
* On-screen keyboard UI
* Local storage streak tracking
* Reveal animations
